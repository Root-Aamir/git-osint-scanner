# 🛡️ Git OSINT Scanner

A powerful full-stack cybersecurity tool designed to monitor and scan local Git repositories for sensitive data leaks. It helps developers find accidentally committed secrets like passwords, API keys, and sensitive tokens.

## 🚀 Features
- **Real-time Commit Scanning:** Scans recent commit history using optimized regex patterns.
- **Modern Dashboard:** Dark-themed UI built with Next.js and Tailwind CSS.
- **Backend Engine:** Built with Laravel for high-performance file system operations.
- **Security Focused:** Detects hardcoded credentials across the entire repo history.

## 🛠️ Tech Stack
- **Frontend:** Next.js, Tailwind CSS, Lucide Icons
- **Backend:** PHP (Laravel)
- **Database:** MySQL
- **Environment:** Node.js, Git

## 📦 Installation
1. Clone the repository: `git clone https://github.com/Root-Aamir/git-osint-scanner.git`
2. Set up the Backend (Laravel): `composer install` & `php artisan serve`
3. Set up the Frontend (Next.js): `npm install` & `npm run dev`
